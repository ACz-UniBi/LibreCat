NAME

LibreCat::App::Catalogue::Route::person - handles person settings

PREFIX /librecat/person

All person settings are handled within the prefix '/person'.

GET /preference

User edits the preferred citation style and sorting for his own publication list.

POST /author_id

User adds author identifiers to db (e.g. ORCID). These will be displayed on author's profile page.

POST /set_language

User can choose default language for the librecat backend "en" -> English - default "de" -> German

POST /affiliation

User edits his affiliation. Will be displayed if you opens new publication form.
