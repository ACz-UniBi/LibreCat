NAME

LibreCat::Cmd::audit - manage librecat audit messages

SYNOPSIS

    librecat audit list [<RECORD-ID>]
    librecat audit get <AUDIT-ID>

    An 'audit' worker should be up and running to
    store messages:

    Hint:

    bin/librecat worker audit start --workers 1 --supervise
